# Lawyer Baba Chatbot

## Project Overview
Lawyer Baba Chatbot is a Python-based chatbot that provides information about a lawyer, case details, court information, hearing dates, judges, fees, and contact details.

This project is designed for beginners who want to learn Python chatbot development using conditional statements and loops.

---

## Features

- Lawyer Personal Information
- Education Details
- Professional Background
- Daily Routine Information
- Hobbies and Interests
- Future Goals
- Case Information
- Hearing Dates
- Judge Information
- Court Information
- Consultation Fee Details
- Contact Information
- Interactive Chat Interface

---

## Technologies Used

- Python 3
- Conditional Statements (if-elif)
- While Loop
- User Input Handling
